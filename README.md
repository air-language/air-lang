# Air Website

The official website for **Air**, a programming language that compiles directly to native machine code.

This project presents the language itself: its runtime model, syntax direction, documentation, install flow, and core language capabilities across a dedicated multi-page website.

## What This Project Is

This repository contains the web experience around Air, including:

- a landing page focused on the language
- a documentation page for technical content
- an install page for Windows, macOS, and Linux
- language-specific content and translations
- static download assets and route handling for installers

## Project Focus

The site is built to communicate a small set of core ideas clearly:

- **Native machine code**
  Air compiles directly to native output instead of relying on a VM or interpreter.

- **Direct syntax**
  The language aims to reduce unnecessary boilerplate and keep common tasks straightforward.

- **One language across different kinds of work**
  Air is presented as a language for backend services, tools, desktop software, and lower-level systems work.

## Main Sections

### Home

The homepage introduces the language through structured sections, animated presentation blocks, and a visual system built around Air branding.

### Docs

The documentation page is used for language explanation, architecture notes, runtime-related material, and technical reading.

### Install

The install page provides platform-specific entry points for Windows, macOS, and Linux, with direct download routes and command-line setup guidance.

## Language Coverage

The site currently includes content in English, Simplified Chinese, Traditional Chinese, and Russian.

## Purpose

This repository exists to present Air clearly and consistently on the web. It is not a general product shell or a generic landing page template. The focus is the language itself: what it is, how it works, where it is used, and how people can read the docs or install it.
